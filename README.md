# n-radio-downloader (WIP)

Save NHK Radio shows locally.
It consists of a simple, less depenency Node.js library and an Electron app.
Electron app provides portability and GUI.

## installation

requires Node.js

```bash
npm install
```

## build library

```bash
npm run build -w downloader
```

## development

```bash
npm run start -w app
```
