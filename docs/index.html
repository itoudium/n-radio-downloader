n-radio-downloader

NHKラジオ（らじるらじる）の番組を保存するPC用アプリです。
聞き逃し配信のWebページを参照し、公開されている番組を自動的にダウンロードします。
本アプリは非公式で、動作保証はありません。
公式の利用規約をご確認の上、ご利用ください。

機能

  • ダウンロードしたい番組をフォローします
  • 自動的に更新をチェックします
  • 自動的にダウンロードします
  • 聞き逃し配信ページに公開されている限り、過去に遡ってダウンロードします
  • ダウンロード先のフォルダを設定できます
  • プレイヤー機能はありません
  • リアルタイム録音ではありません

ダウンロード

Releases から最新版をダウンロードしてください。

対応環境: Windows x64, macOS arm64
